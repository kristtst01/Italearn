#### Unit 29: Conditional Sentences (If Clauses)
**Grammar:**
- [ ] Type 1 — real/possible: se + presente, futuro/presente (se piove, resto a casa)
- [ ] Type 2 — unreal/hypothetical present: se + congiuntivo imperfetto, condizionale presente (se avessi tempo, viaggerei)
- [ ] Type 3 — unreal past: se + congiuntivo trapassato, condizionale passato (se avessi studiato, avrei passato l'esame)
- [ ] Mixed conditionals

**Vocabulary:**
- [ ] Hypothetical situations, regrets, possibilities
