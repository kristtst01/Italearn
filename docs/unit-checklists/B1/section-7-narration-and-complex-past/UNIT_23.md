#### Unit 23: Imperfetto vs Passato Prossimo
**Grammar:**
- [ ] Passato prossimo = completed action, specific time
- [ ] Imperfetto = ongoing, habitual, background
- [ ] Narrative structure: imperfetto for scene-setting, passato prossimo for events
- [ ] "Mentre + imperfetto, passato prossimo" pattern
- [ ] Meaning change verbs (sapere: sapevo = I knew vs. ho saputo = I found out; conoscere: conoscevo = I was acquainted vs. ho conosciuto = I met)
